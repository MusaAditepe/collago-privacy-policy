1. 소개

Collago - 친구들과 함께 만들기("우리", "저희", "앱")는 사용자의 개인정보 보호를 약속합니다. 이 개인정보처리방침은 앱을 사용할 때 개인정보를 어떻게 수집, 사용, 저장, 공유하는지 설명합니다.

2. 수집하는 정보

2.1 개인정보

  • 계정 정보: 이름, 성, 이메일 주소, 사용자명

  • 연락처 정보: 전화번호 (선택사항)

2.2 사용 정보

  • 콘텐츠: 생성된 콜라주, 공유된 사진, 댓글
  • 소셜 상호작용: 친구 목록, 그룹 멤버십, 메시징

2.3 기술 정보

  • 기기 정보: 기기 유형, 운영체제, 앱 버전
  • 위치 정보: 일반적인 위치 데이터 (선택사항)
  • 로그 정보: IP 주소, 브라우저 유형, 접근 시간
